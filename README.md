## 📌 Projetos em Destaque

### 🏦 [Digital Bank API](https://github.com/JosemarJR-Dev/digital-bank-api)

API RESTful de banco digital desenvolvida com Spring Boot, simulando o core de transações financeiras com foco em resiliência e concorrência.

* Implementação de transferências transacionais com `@Transactional`.
* Controle de concorrência com lock pessimista para evitar inconsistência financeira.
* Gestão de movimentações financeiras de débito e crédito.
* Disparo de notificações pós-commit após transferências concluídas.
* Testes unitários, teste concorrente, Swagger, Docker e pipeline CI com GitHub Actions.

### 📊 [Spring Boot Observability Logs - Elasticsearch & Kibana](https://github.com/JosemarJR-Dev/java-spring-boot-observability-logs-elasticsearch-kibana)

Projeto de observabilidade backend desenvolvido com Java e Spring Boot, focado em logs estruturados, correlação de requisições e visualização centralizada com Elasticsearch e Kibana.

* API REST simulando operações de pedidos para geração de logs reais.
* Implementação de `requestId` por requisição para rastreabilidade ponta a ponta.
* Logs estruturados em JSON utilizando Logback e MDC.
* Envio de logs para Elasticsearch por meio de appender customizado.
* Visualização e análise dos logs no Kibana.
* Tratamento global de exceções com respostas padronizadas.
* Ambiente completo com Docker Compose, incluindo aplicação Spring Boot, Elasticsearch e Kibana.
* Uso de Spring Boot Actuator para endpoints de saúde e métricas.
